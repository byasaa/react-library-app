# React x Redux - Library App

An application to make online book loan transactions based on Web.

## Features

* Login, Register, Logout
* Create, Read, Update, Delete (CRUD)
* Borrow and Return Book
* Search, Paginate, Sort
* Upload Image
* Admin Access Menu
